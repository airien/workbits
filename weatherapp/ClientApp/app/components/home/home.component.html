Oppgave 2: Fra kaldt til varmt

  1. Forandre WeatherDataController slik at den returnerer riktig beskrivelse for temperatur:
    • temperatur < -10 -> Freezing
    • temperatur < -5 -> Bracing
    • temperatur < 0 -> Chilly
    • temperatur < 5 -> Cool
    • temperatur < 10 -> Mild
    • temperatur > 10 && temperatur <= 20 -> Warm
    • temperatur > 20 -> Balmy
    • temperatur > 25 -> Hot
    • temperatur > 30 -> Sweltering
    • temperatur > 35 -> Scorching
  2. Endre tekst på fetch data til Show weather
  3. Legg til en knapp i Show weather som laster siden på nytt
  4. Hvordan kan vi få det til å se penere ut?
  5. Hvordan kan vi legge til faktisk vær i oslo siste 5 dager?
  6. Hva er de beste måtene å teste denne koden på?
